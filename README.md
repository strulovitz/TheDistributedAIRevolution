## About This Book

This book is for everyone. You do not need to be a programmer, an engineer, or a tech expert. If you can read, you can understand this book — and by the end of it, you will understand how ordinary computers can work together as a powerful AI network, and how you can be a part of it.

Whether you want to earn money from your idle computer, run a business coordinating AI work, get affordable AI processing for your company, or set up a private AI system inside your organization where sensitive data never leaves the building — this book will show you how.

The entire platform described in this book is **free and open source**. The code is on GitHub. The book is free. Everything you need to get started is here.

## The Platform

This book describes a real, working, tested platform. The code is free and open source:

- **[BeehiveOfAI](https://github.com/strulovitz/BeehiveOfAI)** — The website and marketplace hub (Flask, Python)
- **[HoneycombOfAI](https://github.com/strulovitz/HoneycombOfAI)** — The desktop client software with native GUI (PyQt6, Python)

Tested on Windows and Linux. Proven across the real internet. Deployed via Cloudflare Tunnel. Supports 5 AI backends: Ollama, LM Studio, llama.cpp server, llama.cpp Python, and vLLM.
